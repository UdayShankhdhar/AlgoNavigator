# AlgoNavigator
AlgoNavigator is an interactive algorithm visualizer that helps users understand data structures and algorithms through real-time animations and step-by-step execution.
# AlgoNavigator 🚀

## Overview
AlgoNavigator is an interactive platform to visualize algorithms in real-time.

## Features
- Sorting Visualizer (Bubble, Merge, Quick)
- Searching Algorithms
- Step-by-step execution
- Clean UI

## Tech Stack
- Frontend: React.js / HTML / CSS / JS


## Installation
git clone <repo-link>
cd project
npm install
npm start
